# 새로운마케팅 (saerounmarketing.com) 개발 로드맵

수의사가 직접 운영하는 동물병원 전문 마케팅 에이전시 사이트의 고도화 로드맵

## 개요

새로운마케팅은 동물병원 원장님을 대상으로 마케팅 서비스를 홍보하고 상담 리드를 수집하는 랜딩페이지입니다. 현재 기본 랜딩페이지가 완성된 상태이며, 다음 방향으로 고도화를 진행합니다:

- **전환율(CVR) 향상**: 감사 페이지, 플로팅 CTA, 단계별 폼 등으로 리드 전환율 극대화
- **신뢰도 강화**: 고객 후기 슬라이더, 성과 지표, 파트너 로고, 콘텐츠 샘플 갤러리
- **콘텐츠 마케팅 인프라**: 블로그/아티클 시스템, 뉴스레터, RSS 피드
- **자동화 & CRM**: Make 웹훅, 노션 CRM, 자동 이메일 시퀀스, 채팅 위젯
- **성과 분석 고도화**: GA4 맞춤 대시보드, 네이버 애널리틱스, A/B 테스트, 히트맵

## 기술 스택

- **프레임워크**: Next.js 16 (App Router), React 19, TypeScript
- **스타일링**: Tailwind CSS 4
- **이메일**: EmailJS (@emailjs/browser)
- **분석**: GA4 (G-BXZJX18Q1M)
- **SEO**: JSON-LD, sitemap.ts, robots.ts, OG 태그
- **배포**: Vercel (추정)

## 현재 완료된 기능

- 랜딩페이지 9개 섹션 (Hero, Story, Problem, Solution, Services, Results, FAQ, Contact, Footer)
- EmailJS 기반 상담 신청 폼 (이름, 병원명, 연락처, 고민 사항)
- Make(Integromat) 웹훅 전송 API 라우트 (`/api/contact`) - 환경변수 미설정 시 스킵
- GA4 스크립트 삽입
- SEO 최적화 (JSON-LD 구조화 데이터, 사이트맵, robots.txt, OG/Twitter 메타태그)
- 네이버/구글 소유권 검증 메타태그
- 반응형 디자인 (Tailwind CSS)

## 개발 워크플로우

1. **작업 계획**
   - 기존 코드베이스를 학습하고 현재 상태를 파악
   - 새로운 작업을 포함하도록 `ROADMAP.md` 업데이트
   - 우선순위 작업은 마지막 완료된 작업 다음에 삽입

2. **작업 생성**
   - 기존 코드베이스를 학습하고 현재 상태를 파악
   - `/tasks` 디렉토리에 새 작업 파일 생성
   - 명명 형식: `XXX-description.md` (예: `001-thank-you-page.md`)
   - 고수준 명세서, 관련 파일, 수락 기준, 구현 단계 포함
   - API/비즈니스 로직 작업 시 "## 테스트 체크리스트" 섹션 필수 포함
   - 이전 완료된 작업 파일을 참조 예시로 활용

3. **작업 구현**
   - 작업 파일의 명세서를 따름
   - 기능과 기능성 구현
   - API 연동 및 비즈니스 로직 구현 시 Playwright MCP로 테스트 수행
   - 각 단계 후 작업 파일 내 단계 진행 상황 업데이트
   - 각 단계 완료 후 중단하고 추가 지시를 기다림

4. **로드맵 업데이트**
   - 로드맵에서 완료된 작업을 체크 표시로 업데이트
