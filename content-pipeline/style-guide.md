# 새로운마케팅 글쓰기 스타일 가이드

마지막 업데이트: 2026-03-30 (마케터 일지 시리즈 분석 기반)

## 핵심 원칙
**"전문가가 하는 친구의 조언"**
수의사 출신 마케터가 실제 경험과 데이터를 바탕으로 원장님들에게 편하게 조언하는 톤.
무겁지 않게, 자조적 유머와 친근함으로 접근하되, 냉철한 현실 분석이 뒤따름.

## 문체/톤
- **구어체 반말 기반** — "근데", "그래서", "그럼" 자주 사용
- **1인칭 중심** — "나는", "내가", "내 생각에는"으로 개인 책임감 있게 주장
- **짧은 단문 위주** — "다."로 끝나는 문장이 많아 명확하고 강렬함
- **감정 표현 허용** — ㅠㅠ, (?), ... 등으로 솔직한 감정 드러냄
- **자조적 유머** — 무거운 주제도 가볍게 처리

## 구조 패턴
**시작**: 강한 질문이나 자극적 명제 → 독자 주의 집중
**본론**: 주장 → 근거/사례 → 결론 반복, 원장님 실제 목소리 인용
**마무리**: 따뜻한 격려 + 행동 유도 (연락주세요, 시작해보시면 좋겠다)

## 특징적 표현
- "어차피 ~~다" (현실 수용)
- "~~할수록 ~~할수밖에 없다" (악순환/선순환)
- "내가 본 바로는", "내 생각에는" (개인 견해)
- "싫어도 그게 현실이다" (냉철한 수용)
- 배민, 애플, 강남 등 실제 사례 인용

## 금지 표현
- "~하는 것이 중요합니다" (X) → "그게 중요하다" (O)
- "~를 통해 효과를 거둘 수 있습니다" (X)
- "많은 분들이" (X) → "원장님들 보면" (O)
- 해시태그 남발 (X)
- 거창한 학술용어 (X) → 있으면 바로 쉽게 재정의
- 과도한 수식어 "엄청", "매우" 남발 (X)
- "~인 것 같습니다" 존댓말 (X)

## 실제 예시 문장
1. "시장이 커지는 속도보다 수의사가 공급되는 속도가 더 빠르다. 그래서 동물병원도 많이 망한다는 소리가 들려온다."
2. "나는 가격으로 브랜딩한다면 오히려 비싸게 가라고 말한다. 저렴함은 1등만 살아남는다. 하지만 비쌈은 다르다."
3. "배민도 처음엔 그냥 배달 앱이었다. 근데 지금은? 광고비 내야 되고, 수수료 내야 되고."
4. "싫어도 그게 현실이다. 원장님들도 시작해보시면 좋겠다."
5. "마케팅이 뭐 해주는 대? 효과 있나?"

## 최근 성과 데이터
- 잘 되는 패턴: 마케터 일지 (개인 경험 기반) — SEO 정보글 대비 8~10배 조회수
- 안 되는 패턴: "~하는 방법", "~가 중요한 이유" 형식의 정보글
- 최고 성과 글: 일지 #6 (41 views/month), 일지 #1 (38 views/month)
