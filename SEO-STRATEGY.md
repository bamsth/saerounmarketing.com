## Phase 0 — 네이버 긴급 복구 ✅ 완료 (2026-04-02)

### 왜 먼저인가
네이버 색인 0 = 네이버 검색에서 사이트가 존재하지 않음. 이 상태에서 다른 작업을 해봤자 의미 없음.

### Task 0-1: 네이버 서치어드바이저 HTTPS 재등록 (수동) ✅
1. searchadvisor.naver.com 접속
2. 웹마스터 도구 → 사이트 관리 → 사이트 등록
3. `https://saerounmarketing.com` 입력 (http 아님, www 아님)
4. 소유권 확인: HTML 메타태그 방식 (layout.tsx에 이미 naver-site-verification 있음)
5. 기존 `http://` 버전 등록 있으면 삭제

### Task 0-2: 수집제한 해제 (수동) ✅
1. 서치어드바이저 → 해당 사이트 → 설정
2. 트래픽 제한 요청 → "수집제한 설정하지 않음" 체크
3. 수집주기 → "빠르게" 설정

### Task 0-3: 사이트맵/페이지 수집 요청 (수동) ✅
1. 서치어드바이저 → 요청 → 사이트맵 제출 → `https://saerounmarketing.com/sitemap.xml`
2. 요청 → 웹페이지 수집 → 메인 URL 직접 제출

### Task 0-4: robots.ts 네이버 봇 명시 허용 확인 (코드) ✅
```ts
rules: [
  { userAgent: 'Yeti', allow: '/' },
  { userAgent: '*', allow: '/' },
]
```

**완료 기준**: 서치어드바이저에서 사이트 상태 "정상" + 색인 수 증가 시작

## Phase 1 — 기술 SEO 기반 강화 ✅ 완료 (2026-04-02)

### Task 1-1: 구조화 데이터 보강
현재: `ProfessionalService` + FAQ 3개

추가할 것:
```json
"founder": {
  "@type": "Person",
  "name": "김범수",
  "hasCredential": {
    "@type": "EducationalOccupationalCredential",
    "credentialCategory": "수의사 면허",
    "recognizedBy": { "@type": "Organization", "name": "농림축산식품부" }
  },
  "knowsAbout": ["수의학", "동물병원 경영", "디지털 마케팅", "네이버 SEO"]
}
```
- FAQPage 스키마 → 현재 3개 → 5개로 (컴포넌트 개수와 일치)
- 블로그 글 발행 시 Article 스키마 + author Person 연결 체계 구축

### Task 1-2: llms.txt 최적화 (GEO)
`/public/llms.txt` 내용 보강 — AI 검색(ChatGPT, Perplexity, 구글 AI) 인용 최적화

### Task 1-3: 신뢰 페이지 추가 (E-E-A-T)
- `/privacy` 개인정보처리방침 (구글 E-A-T 필수)
- `/about` 김범수 수의사 프로필 전용 페이지 (Person 스키마 연결)
- Footer에 사업자등록번호 추가

### Task 1-4: GA 스크립트 성능 최적화
next/script strategy="afterInteractive" 적용 (현재 렌더링 차단 가능성)

**완료 기준**: PageSpeed Insights 90+ / 구조화 데이터 검증 통과

## Phase 2 — 블로그 시스템 구축 + 콘텐츠 SEO ✅ 완료 (2026-04-02)

### 왜 블로그가 필수인가
"동물병원 마케팅" 키워드는 정보성+상업성 복합 의도 → 구글은 깊이 있는 콘텐츠를 가진 사이트 우선 노출. 단일 랜딩 페이지로는 구글 1페이지 진입 구조적 불가.

### Task 2-1: /blog 라우트 구현
```
app/
  blog/
    page.tsx          # 블로그 목록
    [slug]/
      page.tsx        # 개별 포스트 (generateMetadata + Article 스키마)
```
- 각 포스트에 generateMetadata (타이틀, 설명, OG 자동 생성)
- Article 스키마 JSON-LD 자동 삽입
- 저자 박스 컴포넌트 (김범수 수의사 프로필)
- 사이트맵에 블로그 URL 자동 포함

### Task 2-2: Pillar 콘텐츠 ("2026년 동물병원 마케팅 완벽 가이드")
- 타겟 키워드: `동물병원 마케팅`
- 분량: 5,000자+
- 8개 서브토픽 요약 → 각 블로그 글로 내부 링크

### Task 2-3: 1단계 블로그 4편

| 파일 | 타겟 키워드 | 월 검색량 |
|------|------------|-----------|
| 01-플레이스-최적화.md | 동물병원 네이버 플레이스 | 100~200 |
| 08-개원-마케팅.md | 동물병원 개원 마케팅 | 50~100 |
| 04-광고비-적정-금액.md | 동물병원 광고비 | 50~100 |
| 03-홍보-전단지-vs-블로그.md | 동물병원 홍보 방법 | 100~200 |

각 포스트 필수 요소:
- H1: 타겟 키워드 포함
- 도입부에 핵심 요약 (AI 인용 최적화)
- 통계/수치 포함 (AI 인용률 22% 증가)
- FAQ 섹션 (FAQPage 스키마)
- "수의사의 관점에서" 직접 인용문 (AI 인용률 37% 증가)
- 저자 박스
- Pillar 포스트로 내부 링크
